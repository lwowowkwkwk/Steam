Steam Client for Windows 7



This repository contains the source code for a compatibility launcher that restores full functionality to the official Steam client on Windows 7 / Server 2008 R2. Valve dropped support officially, but the underlying Win32 API is still compatible.



Modifications:

\-add support windows 7



Build instructions: Visual Studio 2019 with v141\_xp toolset. Just compile and place in /bin.
